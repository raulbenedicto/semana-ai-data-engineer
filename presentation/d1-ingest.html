MCP · MODEL CONTEXT PROTOCOL

O protocolo universal que conecta agentes ao mundo real

Descoberta Automática

O cliente lista todas as ferramentas disponíveis no server via tools/list. Zero configuração manual de capabilities.

Na prática:

claude > tools/list → execute_sql, semantic_search, create_pr...

Protocolo Padronizado

JSON-RPC 2.0 sobre stdio ou SSE. Qualquer linguagem, qualquer plataforma, mesma interface universal.

Na prática:

{ "jsonrpc":"2.0", "method":"tools/call", "params":{...} }

Segurança por Design

Cada tool call precisa de aprovação explícita. O agente pede, o protocolo media, o usuário controla.

Na prática:

Allow tool execute_sql? [Y/n] → usuário aprova cada ação

MCP transforma Claude Code de assistente de texto em plataforma conectada. Um protocolo aberto, padronizado pela Anthropic, que qualquer server pode implementar — e qualquer client pode consumir.

CORROMPEM O PIPELINE

Com Pydantic

Cada campo tem tipo e constraint — dados inválidos nunca passam.

from pydantic import BaseModel, Field

from typing import Literal

 

class Order(BaseModel):

  order_id: str

  qty: int = Field(ge=1, le=100)

  total: float = Field(ge=0)

  payment: Literal[

    'pix','credit_card','boleto']

 

order = Order(**raw)  # Validado!

# ValidationError se inválido

Cada entidade, um contrato

models/order.py

from pydantic import BaseModel, Field, field_validator

from typing import Literal

from datetime import datetime

import uuid

 

class Order(BaseModel):

  order_id: str = Field(

    default_factory=lambda: str(uuid.uuid4()))

  customer_id: str

  product_id: str

  qty: int = Field(ge=1, le=100)

  total: float = Field(ge=0)

  status: Literal[

    'delivered','shipped',

    'processing','cancelled']

  payment: Literal[

    'pix','credit_card','boleto']

  created_at: datetime

 

  @field_validator('total')

  @classmethod

  def total_must_be_reasonable(cls, v):

    if v > 50000:

      raise ValueError('Total acima do limite')

    return round(v, 2)

Field Constraints

Limites numéricos declarativos: ge=maior-ou-igual, le=menor-ou-igual.

Na prática:

qty: int = Field(ge=1, le=100)

Literal Types

Enum implícito — só aceita os valores listados, qualquer outro rejeita.

Na prática:

payment: Literal['pix','credit_card','boleto']

@field_validator

Lógica customizada de validação por campo — Pydantic v2 usa @field_validator + @classmethod.

Na prática:

total > 50000 → ValueError

ValidationError

Falha explícita com mensagem clara — nunca corrompe silenciosamente. Mostra campo, valor e regra violada.

Na prática:

1 validation error: total > 50000
